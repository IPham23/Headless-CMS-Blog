# Headless CMS Blog

A demo project built to showcase my headless WordPress setup using React and TypeScript.

Tech Stack:
- WordPress (backend CMS)
- WordPress REST API
- React
- TypeScript
- Tailwind CSS

Overview:
WordPress is used purely as a content management system. The frontend is a separate React application that fetches posts through the REST API.

What this project demonstrates:
- Fetching and rendering dynamic content from WordPress
- Structuring API calls in TypeScript
- Strong typing for API responses
- Decoupled CMS architecture

How it works:
1. Content is managed in WordPress.
2. React fetches data using REST endpoints.
3. The UI renders content dynamically.

How to run locally:
npm install
npm run dev
